Reference

Search syntax help

  • & = match both terms
  • | = also works for combined matching
  • ! = exclude a term
  • , : ; = OR operator
  • - = numeric range
  • + = evolutionary line search

Examples

  • grass&evolve
  • !defender
  • electric,@weather
  • cp2000-
  • +dragonair
